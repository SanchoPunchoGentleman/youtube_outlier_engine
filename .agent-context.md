# SENTINEL-V: AGENT OPERATIONAL CONTEXT

## Project Identity
- **Name**: Sentinel-V YouTube Research Engine
- **Mission**: Identify 3x+ view-to-sub outliers across 1,000 niches to extract structural packaging DNA.
- **Tone**: Professional, B2B, minimalist, high-authority.
- **Hardware**: MacBook Air M1 (8GB RAM). 

## Technical Guardrails
- **Language**: Strictly English-market (US, UK, AU, CA).
- **Ethics**: Zero-tolerance for content involving alcohol, pork, interest-based finance, or obscenity.
- **Architecture**: Python search engine -> n8n orchestration -> Obsidian Graph visualization.
- **API Strategy**: Maximize quota efficiency (use `playlistItems` over `search` where possible).

## Repository Structure
- `/src`: Core logic (scraper, analyzer, bridge).
- `/docs`: Google Audit documentation, Privacy Policy, README.
- `/config`: User inputs (niches, manual channel overrides).
- `/data`: CSV outputs and temporary logs.
