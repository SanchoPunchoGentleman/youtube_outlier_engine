# Sentinel-V: YouTube Outlier Engine

## 1. Executive Summary
Sentinel-V is a high-authority analytical engine designed to identify "outlier" performance in niche YouTube markets. By calculating the ratio between average view counts and subscriber bases, Sentinel-V pinpoint channels that are punching significantly above their weight class, providing deep insights into emerging content trends and structural B2B patterns.

## 2. Technical Architecture
The system operates on a professional-grade local stack designed for high-throughput analysis and persistent knowledge management:
- **Compute**: M1 Mac (Apple Silicon) optimized for Python-based data processing.
- **Orchestration**: Integration with **n8n** for automated search vectors and recursive data flows.
- **Knowledge Base**: **Obsidian** serves as the primary "brain," where extracted patterns and niche saturations are visualized in a linked knowledge graph.

## 3. The Quota Math
To support a comprehensive longitudinal study of the YouTube ecosystem, we are requesting a quota extension to 5,000,000 units.
- **Market Scope**: 1,000+ distinct niche markets.
- **Depth**: Recursive analysis of 50-100 channels per niche.
- **Unit Cost**: Each niche search vector, including channel statistics and video metadata retrieval, consumes approximately 100-500 units.
- **Frequency**: Daily tracking across multiple months to identify growth trajectories.
- **Calculation**: 1,000 niches x 100 unit base cost x 50 iterations = 5M unit requirement.

## 4. Ethical Filter
Sentinel-V is built with integrated ethical guardrails to ensure high-authority, professional research:
- **Exclusion Lists**: Automatic filtering of content related to alcohol, adult interest, obscenity, and low-signal "hype" niches.
- **US/EN Focus**: Strictly localized to US-en markets to maintain data consistency and professional relevance.
- **Non-Invasive**: No user-level data is accessed; the system operates entirely on public metadata.
